Early Access Game

Get instant access and start playing; get involved with this game as it develops.

Note: This Early Access game is not complete and may or may not change further. If you are not excited to play this game in its current state, then you should wait to see if the game progresses further in development. Learn more

What the developers have to say:

Why Early Access?

“Kingdomfall is an indie game developed on a pure enthusiasm of a small group of people. There were many features implemented recently, but to make sure everything would work as expected, game was fully rewritten to a better and stable framework. Some features should be moved from the older version, some should be added.

We want to make sure players will receive all the features only at the best quality, when they are completely implemented and tested by our team.”

Approximately how long will this game be in Early Access?

“Road to full release can take around two years from now, depending on many factors.”

How is the full version planned to differ from the Early Access version?

“We plan to add more characters, maps, game modes, additional character customization, including appearance, weapons and armor. We are also planning to add siege vehicles as well as environmental destruction. Some maps can contain a lot of water, so we want to add player-controlled Drakkars. Various types of NPCs might also be added, especially for certain PvE game types like Raid.

You can check how some of the features mentioned above might work on the Kingdomfall YouTube page.”

What is the current state of the Early Access version?

“The current state of the early-access version contains the following features:
1. 9 Unique characters with different equipment, skills and abilities.
2. Basic combat system, with class differences for specific characters.
3. Most of the weapon types.
4. Maps: Lost Camp, Borderlands, Arena.
5. Game Modes: Team Deathmatch, Control, Freeplay.
6. Host/Join from the game or a Linux Dedicated Server for download. (Requires additional setup)
7. Emotes.”

Will the game be priced differently during and after Early Access?

“While remaining in early access, the game will be available at half price.”

How are you planning on involving the Community in your development process?

“One of the main reasons to launch Kingdomfall in early access is to let players play it as early as possible. We want to make sure the combat system works as expected and that the game is fun to play. We will try to pay attention to all the feedback the game would receive to improve the game.”

About This Game

Imagine yourself in a world with mighty battles and powerful magic, where deadly fireballs fly over your head burning everything to a complete hell. Where drakkars full of soldiers attack from the sea, giant trebuchets destroy castle walls, burying many warriors under the debris. All of that while the are simple soldiers, going through mud, fire, blood, trying to fight, survive and even win the battle. You are one of those soldiers. That’s what the Kingdomfall is about.

One of the most advanced player-skill based combat system which consists of simple attacks, strong attacks, dodges, counter-attacks, shield strikes, many types of melee and ranged weapons, from bows to deadly magic - it is designed to not limit you in any possible way, if you're skilled enough you will have you're chance to win.

IMPORTANT NOTE The game right now is in early access and will include only part of the future features to ensure everything works as expected. You can take a look at how most of the upcoming features may potentially work on the official YouTube page. They will be moved to the release version one by one when ready. Because of that the game will be available here at half price until full release.

Mature Content Description

The developers describe the content like this:

The game contains combat with melee weapons like swords, spikes, bows, fire magic. The effects are visualized with blood and voice lines. No gore is presented.
